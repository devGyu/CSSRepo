CSS 기본 선택자

스타일 규칙은 태그&웹 문서에 적용이 가능하다. 이때 선택자는 스타일 규칙이 어떤 부분에 적용이 될것인지를 알려주는것

전체 선택자

* { 속성 : 값; .... }

타입 선택자

태그명 { 속성 : 값; .... }

클래스 선택자

.클래스명 { 속성 : 값; .... }

아이디 선택자

#클래스명 { 속성 : 값; .... }

그룹 선택자

선택자1, 선택자2 { 속성 : 값; .... }

캐스케이딩(cascading) : 계단식으로 적용되는, 우선순위가 있는...

즉, CSS는 웹 요소에 둘 이상의 스타일을 적용할때 우선순위에 따라 적용할 스타일을 결정한다.
